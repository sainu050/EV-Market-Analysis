
# Electric Vehicle Market Analysis and Visualization

## Overview
This project analyzes the Indian Electric Vehicle market using data analytics and visualization techniques.

## Objectives
- Analyze EV sales growth.
- Identify top manufacturers.
- Study vehicle categories.
- Analyze charging infrastructure.
- Study EV manufacturer distribution.

## Technologies Used
- Python
- Pandas
- NumPy
- Matplotlib
- Seaborn
- Jupyter Notebook

## Key Findings
- Ola Electric Technologies Pvt Ltd is the leading EV manufacturer.
- Two-wheelers dominate the EV market.
- Maharashtra has the highest number of EV manufacturers.
- EV adoption has grown rapidly after 2020.

## Project Structure
- Data Cleaning
- Exploratory Data Analysis
- Visualization
- Insights
- Conclusion
  
# EV-Market-Analysis
Data Analytics and Visualization project analyzing the Indian Electric Vehicle market using Python, Pandas, Matplotlib, and Seaborn.
